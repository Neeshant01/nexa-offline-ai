# Nexa Offline AI

Multi-module Android project for an offline-first AI assistant experience focused on productivity, local workflows, and modular feature development.

## Stack

- Android
- Kotlin
- Gradle multi-module architecture

## Modules

- app shell
- AI runtime layer
- domain/data/core modules
- feature modules for chat, notes, reminders, settings, and voice

## Status

- public project
- active prototype
